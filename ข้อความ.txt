                <option value="ศิลปกรรมเด็กเเละเยาวชนเเห่งชาติ">ศิลปกรรมเด็กเเละเยาวชนเเห่งชาติ</option>

    <script>
        // Data Management
        let database = JSON.parse(localStorage.getItem('flg_db')) || [
            { id: 1, url: 'https://images.unsplash.com/photo-1549490349-8643362247b5', title: 'Line of Sight', cat: 'Minimal Arts', featured: true },
            { id: 2, url: 'https://images.unsplash.com/photo-1579783902614-a3fb3927b6a5', title: 'Emotional Grid', cat: 'ศิลปกรรมเด็กเเละเยาวชนเเห่งชาติ', featured: true },
            { id: 3, url: 'https://images.unsplash.com/photo-1550684848-fac1c5b4e853', title: 'Cyber Soul', cat: 'Digital Art', featured: false }
        ];

        function showPage(pageId) {
            document.querySelectorAll('.page').forEach(p => p.classList.remove('active'));
            document.getElementById(pageId).classList.add('active');
            window.scrollTo({top: 0, behavior: 'smooth'});
        }

        function renderAll() {
            const homeGrid = document.getElementById('featuredArea');
            const mainGrid = document.getElementById('mainGallery');

            homeGrid.innerHTML = database.filter(a => a.featured).map(art => createCard(art)).join('');
            mainGrid.innerHTML = database.map(art => createCard(art)).join('');
            
            localStorage.setItem('flg_db', JSON.stringify(database));
        }

        function createCard(art) {
            return `
                <div class="art-card">
                    <img src="${art.url}" alt="${art.title}" loading="lazy">
                    <div class="art-card-info">
                        <p style="font-size: 0.7rem; color: #86868b; text-transform: uppercase; letter-spacing: 1px;">${art.cat}</p>
                        <h4 style="font-size: 1.2rem; margin-top: 5px; font-weight: 600;">${art.title}</h4>
                        <button onclick="deleteArt(${art.id})" style="background:none; border:none; color:#ff3b30; font-size: 0.7rem; cursor:pointer; margin-top: 15px; font-weight: 500;">[ Remove Work ]</button>
                    </div>
                </div>
            `;
        }

        function filterGallery(category) {
            const btns = document.querySelectorAll('.cat-btn');
            btns.forEach(btn => {
                const btnText = btn.innerText.trim().toLowerCase();
                const targetText = category.toLowerCase();
                btn.classList.toggle('active', btnText === targetText || (category === 'All' && btnText === 'all works'));
            });

            const mainGrid = document.getElementById('mainGallery');
            if (category === 'All') {
                mainGrid.innerHTML = database.map(art => createCard(art)).join('');
            } else {
                mainGrid.innerHTML = database.filter(a => a.cat === category).map(art => createCard(art)).join('');
            }
        }

        function toggleModal() {
            const m = document.getElementById('modal');
            m.style.display = (m.style.display === 'flex') ? 'none' : 'flex';
        }

        function saveArt() {
            const url = document.getElementById('artUrl').value;
            const title = document.getElementById('artTitle').value;
            const cat = document.getElementById('artCat').value;
            const featured = document.getElementById('isFeatured').checked;

            if(!url || !title) return alert('Please fill in both Image URL and Artwork Name');

            database.unshift({ id: Date.now(), url, title, cat, featured });
            renderAll();
            toggleModal();
            
            document.getElementById('artUrl').value = '';
            document.getElementById('artTitle').value = '';
            document.getElementById('isFeatured').checked = false;
        }

        function deleteArt(id) {
            if(confirm('Are you sure you want to remove this artwork?')) {
                database = database.filter(a => a.id !== id);
                renderAll();
            }
        }

        renderAll();
    </script>
